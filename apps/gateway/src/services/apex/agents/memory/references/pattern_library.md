# Pattern Library
## Pattern Types
1. star_injury_overpriced: Star player questionable + market prices YES high → NO edge
2. weather_ensemble_consensus: 28+/31 GFS members agree → strong directional trade
3. btc_momentum_reversal: RSI extreme + 3 consecutive same-direction 15m → reversal
4. new_listing_mispriced: New market < 1 hour old + thin book → first-mover edge
5. cross_platform_lag: Kalshi lags Polymarket by > 3% on same event → arb
6. category_regression: Category at extreme win/loss streak → expect reversion
## Decay Rules
- Patterns older than 90 days: weight reduced by 50%
- Patterns older than 180 days: weight reduced by 75%
- Patterns older than 365 days: archived, not used in scoring
