# Polymarket API Reference
## CLOB: Central Limit Order Book on Polygon
## Auth: L1 = EIP-712 signing, L2 = HMAC credentials
## Settlement: USDC on Polygon (chain_id: 137)
## Fee: Most markets fee-free, some categories have taker fees
## Gamma API: gamma-api.polymarket.com (market discovery)
## CLOB API: clob.polymarket.com (order placement)
