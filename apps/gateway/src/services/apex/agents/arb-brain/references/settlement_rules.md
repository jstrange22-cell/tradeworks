# Settlement Rules — Cross-Platform Divergences

## Known Divergences (BLOCK these cross-platform arbs)
- Government shutdown: Polymarket=OPM announcement, Kalshi=24hr actual shutdown
- Election results: Polymarket=AP call, Kalshi=official certification
- Temperature: Different weather station sources possible

## Safe Cross-Platform Categories
- Crypto price thresholds (both use CoinGecko/exchange data)
- Sports outcomes (both use official league results)
- IPO dates (both use SEC filings)

## General Rule
Same-venue arbs have ZERO settlement risk.
Cross-venue arbs need explicit settlement rule verification.
When in doubt, skip.
